# Carrier

A self hosted Gmail REST API

## Usage

Basic GET request. Get all messages from your inbox.

```bash

curl "http://localhost:3000/inbox?email=user@gmail.com&password=mypassword&limit=5"

```

## Params
 
+ email
+ password
+ limit
